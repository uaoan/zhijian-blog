导出后怎么发布？

解压 ZIP，把 _posts 中的文章上传到仓库同名目录；把 assets/images/uploads 中的图片上传到仓库同名目录。提交后 GitHub Pages 会自动重新构建。

所有处理都在当前浏览器完成，不会上传账号密码或图片到第三方服务器。
